# Garden Grace

> A flower shop website project curating nature's beauty in every bouquet.

## Features

- Reset CSS
- Fonts Manual ( Lato )
- Favicons ( favicon.io )
- Form Submit ( formspree.io )
- Not Responsive (Desktop Only)
